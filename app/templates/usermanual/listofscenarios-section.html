List of Scenarios


The list of currently created scenarios can be selected here in order to be customized further and ran on the system.

To select a scenario and display its description follow the steps bellow:

(On the Landing Page the Available Scenarios list is viewable. Each Scenario can be selected to view various details about the selected scenario.)

Once the scenario is selected, on the right side the following information will be displayed:



  • Scenario Name:
    • Name given to the scenario.

  • Scenario Description:
    • Displays the scenario description previously determined by the user in either new scenario or upload sections.

  • Files:
    • Displays the file name(s) that are currently associated with the selected scenario.

  • Machines associated with the selected scenario:
    • This section displays the machines associated with the scenario, the following information is available:
      • Machine Name
      • Machine Type
