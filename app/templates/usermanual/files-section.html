Files


A list of previously uploaded files is populated for the user's convenience, divided into Exploits and Vulnerabilities.

To view the list of files previously uploaded, select the Files button on the Landing Page.

Once the user has selected the Files button a modal will pop-up displaying the following information:



  • Vulnerabilities:
    • *Note: Selecting the file will direct to an edit screen.*
    • The Vulnerabilities tab contains all the vulnerable files that have been previously uploaded.

    • The information provided is:
      • Vulnerability:
        • A list of vulnerable files that have been uploaded.
      • Description:
        • The description of the vulnerability the user has previously written.
      • Actions:
        • The option to delete that specific vulnerable file.

  • Exploits:
    • *Note: Selecting the file will direct to an edit screen.*
    • The Exploits tab contains all the exploit files that have been previously uploaded.

    • The information provided is:
      • Exploit:
        • A list of vulnerable files that have been uploaded.
      • Description:
        • The description of the exploit the user has previously written.
      • Actions:
        • The option to delete that specific exploit file.
  • Import:
    • This button allows you to directly upload a file to a system without creating a scenario. *See the Import section for instructions on how to import a file*

  • Select the "X" on the top right corner of the modal to exit out of the Files Section. This option will take you back to the Landing Page.
