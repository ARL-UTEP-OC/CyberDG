Import


The Import button takes a ZIP file containing JSON files with previously created scenario information, as well as any files that are used in it.

To import a scenario follow the instructions below:

(From the Landing Page Select the Import Button)

Once the Import button is selected you will be prompted to fill in the following options:



  • Scenario Name:
    • Gives the user the option to rename the scenario they wish to import.

  • Browse:
    • Allows the user to select the scenario they wish to import from their computers' local directory or external devices directory.

  • To finalize the scenario import, select the Import button.

  • Once you have selected the Import button, The scenario is displayed in the Available Scenarios list tab on the Landing Page.
